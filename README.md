# Prueba Antigravity - Reto Tablas

## Descripción general
Este proyecto consiste en una página web que presenta un reto basado en tablas utilizando estructura de tablas en HTML.  
Su objetivo es practicar el diseño y organización de contenido en formato tabular, así como mejorar la presentación visual mediante estilos CSS y el uso de interactividad básica con JavaScript.

El sitio permite visualizar información estructurada de manera clara, siguiendo un enfoque práctico orientado al aprendizaje de desarrollo web.
